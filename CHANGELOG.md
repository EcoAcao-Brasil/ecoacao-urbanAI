# Changelog

All notable changes to UrbanAI will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-01

### Added
- Initial release of UrbanAI
- ConvLSTM encoder-decoder architecture for spatiotemporal prediction
- Integration with Tocantins Framework for thermal anomaly detection
- Complete preprocessing pipeline for Landsat data
- Training workflow with early stopping and checkpointing
- Future prediction with uncertainty estimation
- Intervention priority analysis
- GeoJSON and raster output formats
- Command-line interface
- Google Earth Engine export script
- Comprehensive documentation

### Features
- Multi-metric prediction (NDBI, NDVI, NDWI, NDBSI, LST, IS, SS)
- Temporal sequence learning with ConvLSTM
- Automated data preprocessing
- GPU acceleration support
- Configurable via YAML files
- Visualization tools for maps and temporal evolution
